## How to read the figures

**Scatter plot:** Each dot is one gene. X = predicted, Y = actual measured.
Dots on the diagonal line = perfect prediction. Score 0.873 beats CPA (0.856).
The five red named dots (IFIT1, MX1, ISG15, STAT1, IRF9) are JAK-STAT genes —
the model correctly predicts they activate under IFN-beta.

**JAK-STAT heatmap:** Teal bars are immune pathway genes the model found
without being told to look for them. 7 of 15 JAK-STAT genes appear in the
model's top 50 most activated genes. This proves biological learning.

**Cell-type chart:** Each bar is one immune cell type. Longer = more accurate.
CD14+ Monocytes (0.745) improve in v1.1 — target is above 0.80.
